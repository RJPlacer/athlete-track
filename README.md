# athlete-track

## Download for Windows

- Primary download page: https://github.com/RJPlacer/athlete-track/releases/latest
- Direct installer (v1.0.0): https://github.com/RJPlacer/athlete-track/releases/download/v1.0.0/Athlete.Track-1.0.exe

Athlete Track is a JavaFX and SQLite desktop application for sports delegation management, including athlete, official, coach, equipment records, QR workflows, and PDF exports.
